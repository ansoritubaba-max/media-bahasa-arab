# media-ajar-bahasa-arab

Media pembelajaran interaktif yang dirancang untuk memfasilitasi penguasaan keterampilan bahasa Arab (istima’, kalam, qira’ah, dan kitabah) melalui pendekatan kontekstual, berbasis teknologi, dan visual yang menarik guna meningkatkan motivasi serta hasil belajar siswa.

---

## Tentang Proyek

Proyek ini dibangun menggunakan **React + Vite**. 

Untuk menjalankan proyek ini secara lokal:
1. Pastikan Anda sudah menginstal Node.js.
2. Jalankan `npm install` untuk menginstal dependensi.
3. Jalankan `npm run dev` untuk memulai server pengembangan.
